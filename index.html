Proyecto ArcData – Desarrollo de Aplicativo Web para la Fundación Arcángeles

Cargo: Diseñador de Mockups y Desarrollador Web

Fecha: Actualidad

Actualmente, formo parte de un equipo de cuatro personas trabajando en el desarrollo de un aplicativo web para la gestión de citas médicas en la Fundación Arcángeles. Este proyecto busca mejorar la eficiencia en la atención a los pacientes, facilitando el agendamiento de citas, la organización administrativa y el acceso a historiales médicos.

Responsabilidades y aportes clave:

  • Diseño de Mockups: Creación de prototipos visuales en Figma para definir la estructura y experiencia de usuario del aplicativo.
  • Desarrollo Frontend y Backend: Implementación de una interfaz intuitiva y accesible, utilizando HTML, CSS, JavaScript y tecnologías modernas.
  • Base de datos y seguridad: Gestión de la información médica en una base de datos en MySQL, asegurando la privacidad y cumplimiento de normativas de seguridad.

Este proyecto representa un avance tecnológico clave para la fundación, permitiendo a los pacientes gestionar sus citas sin barreras de movilidad y brindando al personal administrativo herramientas para mejorar la calidad del servicio.
